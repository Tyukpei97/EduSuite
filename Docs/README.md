# EduSuite

Модульный учебный проект по C# / .NET 8.

Решение собирает в одном месте несколько учебных программ:

- общий WinForms-лаунчер;
- консольные утилиты (файловый менеджер, шифровщик, синхронизатор папок);
- три GUI-программы по заданию **Task2** (Excel-viewer, график `y = kx + b`, смешивание HEX-цветов).

Проект используется как задание по учебной практике: практика работы с .NET 8, WinForms, консолью, файловой системой, процессами и базовой архитектурой.

---

## Структура решения

```text
EduSuite.sln                – решение Visual Studio

Launcher/                   – WinForms-лаунчер модулей

Modules/
  FileManager.Console/      – консольный файловый менеджер
  Encryptor.Console/        – шифровщик/дешифровщик текстовых файлов
  Sync.Console/             – синхронизатор двух папок (репликация)
  Task2/
    Task2.Core/             – общее ядро для заданий Task2
      ExcelCore.cs          – логика работы с Excel (ClosedXML)
      ...                   – логика графика и смешивания цветов
    ExcelViewer.WinForms/   – GUI-просмотрщик Excel-файлов
    LinePlotter.WinForms/   – GUI для графика y = kx + b
    ColorMixer.WinForms/    – GUI для смешивания HEX-цветов

Docs/                       – методички и отчёты по учебной практике
